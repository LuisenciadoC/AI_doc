# AI_doc
Sistema inteligente de gestión documental empresarial basado en IA, con control de versiones, búsqueda semántica y arquitectura escalable.
Proyecto realizado con los siguientes lenguajes y herramientas:
- Python v3.11.9
- Ollama v0.23.2 (IA)
- Microsoft SQL Server 2025 v17.0.1000.7
- Visual Studio Code v1.119.0
    Extensiones:
    -  Thunder client
